# Book Flow

Book Flow, ReactJS (Vite) ile gelistirilmis bir kitap okuma hedefi takip uygulamasidir.
Bu proje, TODO App mantigini baz alir ve temel CRUD islemlerini icerir.

## Ozellikler

- Ekle: Yeni kitap/hedef ekleme
- Listeleme: Eklenen hedefleri listeleme
- Guncelleme: Kaydi satir icinde duzenleme
- Silme: Kaydi listeden kaldirma
- Tamamlandi durumu: Hedefi isaretleme (checkbox)
- Ozet: Toplam ve biten hedef sayisi

## Teknolojiler

- ReactJS
- Vite
- JavaScript (ES Modules)
- Bootstrap 5
- CSS
